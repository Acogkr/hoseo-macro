# Hoseo Macro

학습 목적으로 개발된 호서대학교 LMS 자동 수강 프로그램입니다.
해당 매크로 사용에 대한 어떠한 책임도 지지 않습니다. 

## 기능
- LMS 자동 로그인
- 미수강 강의 자동 탐색 및 수강
- 백그라운드 실행 (Headless Chrome)
- GUI 제공 (PySide6)

## 실행 방법
1. 필요한 라이브러리 설치:
   ```bash
   pip install -r requirements.txt
   ```
2. 프로그램 실행:
   ```bash
   python hoseo_gui_pyside.py
   ```
